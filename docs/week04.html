深入淺出的解讀

演講最令人印象深刻的是博士將複雜的「地震層析成像」比喻為生活常識。刻意跳過艱澀專有名詞，而是鎖定關鍵原理，讓即便對地物知之甚少的學生也能大致理解地下結構的尺度觀念。

物理核心：F = ma

博士強調所有的正演與逆推問題，其靈魂皆源自「牛頓第二運動定律」。在連續體空間中轉化為應力與位移的數學推斷，這是將理論化繁為簡的極致展現。

層析演算法的邏輯架構

FORWARD PROBLEM 正演問題

給定「模型（地下構造）」，利用物理定律推算出「預測資料（波形、到時）」。這是模擬地震波如何傳遞的過程。

Model → (Physics: F=ma) → Data

INVERSE PROBLEM 逆推問題

從「觀測資料（地震紀錄）」反過來推求「模型參數」。這是一個「病態問題」，因為可能有多種模型都能解釋同一組資料。

Data → (Optimization) → Model
